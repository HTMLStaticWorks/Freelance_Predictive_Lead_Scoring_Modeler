# Scorelytics - Freelance Predictive Lead Scoring Modeler Template

Scorelytics is a high-end, production-ready 13-page HTML template designed specifically for B2B Data Science consultants and Lead Scoring specialists.

## Key Features
- **13 Professional Pages**: Includes 2 Home versions, a complete Admin Dashboard, Blog, Services, and more.
- **Interactive Visuals**: Powered by Chart.js for ROC curves, feature importance, and conversion funnels.
- **ROI Calculator**: Real-time JavaScript widget to demonstrate value uplift to clients.
- **Dark/Light Mode**: Full theme support with system preference detection.
- **RTL Support**: Built-in mirrored layout support for multilingual projects.
- **Bootstrap 5**: Modern, responsive, and easy to customize.

## Getting Started
1. Open `pages/index.html` in your browser to view the main landing page.
2. Customize variables in `assets/css/style.css` to match your branding.
3. Update chart data in `assets/js/charts.js`.

## Tech Stack
- HTML5 / CSS3 (CSS Custom Properties)
- JavaScript (ES6+)
- Bootstrap 5
- Chart.js
- AOS (Animate on Scroll)
- Phosphor Icons
